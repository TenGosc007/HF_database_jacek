# HF_database_jacek
Database of users

## Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:5000
npm start
# Dev Server (Nodemon)
npm run dev
```

### ENV file
``` bash
PORT = 

DATABASE = ''
USER_NAME = ''
PASSWORD = ''
```

### First launch of the base
ORDER IS IMPORTANT!
 - 1 Create first product named NULL
 - 2 Create first user named My/Olek
 - 3 Create next products named Koktail and Koktail_XL
